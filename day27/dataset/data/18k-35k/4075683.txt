【业务背景】
1、极光是业界领先的移动大数据和开发者平台，过去不断开拓各类增值产品和服务，并稳定取得佳绩。从起家的推送业务，到全面的开发者服务、精准营销、数据产品 等，都有非常高的效益，为不同行业的客户带来巨大价值
2、极光调研 是极光具有战略性的新业务，开始以来已经取得市场验证，建于极光的大数据基础和市场优势之上，《极光调研》颠覆固有行业模式，为客户带来同业造不到的价值，也同时为极光带来可观的收益，丰富极光产品体系的生态互补，把极光推到产业链更有利的位置
【工作职责】
1、负责搭建公司的新业务平台《极光调研》，从前端到后端的 Web 全栈架构
2、极光大数据整合、问卷管理/投放、动态报告等功能的搭建、演化、和维护
3、数据算法与存储的设计和优化
4、整合极光其他产品，构建闭环生态，推动公司规模变现业务
【岗位要求】
1、熟悉 Python 开发，熟悉 Django 或其他 Web 框架；有其他相关经验，如熟悉 Java web 架构，也会考虑
2、熟悉 Linux 开发环境，和大型互联网系统的研发和生态
3、了解常用开源开发组件 (Redis、Couchbase、RabbitMQ、Kafka、ElasticSearch、MySQL/PostgreSQL、等)
4、有兴趣学习各种大数据生态相关技术，包括 Hadoop、Spark、Storm、MapReduce、HDFS、HBase、Hive、Kylin 等等
5、能快速学习、灵活解决问题、有上进心和创新思维，对新语言/架构/方法都能很快上手
6、计算机相关专业本科以上学历，3 年以上互联网产品开发工作经验
