一 、 岗位职责 
 1 、 负责 微 门户 产品 （ 用户 1 亿 ， 月活 1000 万 ） 重要 业务 模块 的 前端开发 工作 ； 
 2 、 参与 前端 整体 发展 计划 与 基础 建设 等 ； 
 3 、 参与 推动 与 优化 已有 前端 项目 的 基础架构 与 组件 抽象 ；   
 4 、 针对 新人 、 普通 开发人员 进行 有效 辅导 ， 帮助 其 快速 成长 ； 
 
 二 、 招聘 要求 
 1 、 教育 ： 全日制 大学本科 及 以上 
   2 、 技能 ： 精通   HTML / JS / CSS   等 前端 相关 技术 ； 对 主流 前端 框架 （   React \ Vue \ Angular   等 ） 有 一定 了解 ， 并 至少 对 其中 一种 有 深入 了解 ； 对 服务端 （   Nodejs \ Java \ Python \ Go   等 ） 有 一定 实际 的 开发 经验 ； 拥有 良好 的 推动力 与 主动性 ， 能 主动 发现 问题 并 推进 问题 改进 。 
   3 、 经验 ： 3 年 以上 开发 经验 ， 拥有   2   年 以上 的 前端开发 经验 ； 参与 过 一个 线上 系统 的 创建 、 规划 与 开发 优先 ； 有 Nodejs 开发 经验 优先 。
