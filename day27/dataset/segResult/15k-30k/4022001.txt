职位 描述 ： 
 岗位职责 
 1 、 设计 并 构建 可 交付 的 端 到 端 业务 系统 ， 满足 系统 的 可 运营 需求 ； 
 2 、 构建 灵活 易用 的 Web 系统 ， 对接 后端 AI 平台 ， 并 进行 数据 可视化 ； 
 3 、   对接 客户 技术 和 产品 部门 ， 满足 客户 对 用户界面 简洁 易用 的 要求 。 
 
 岗位 要求 
 1 、 掌握 Web 前端 技术 ， HTML / CSS3 / JS ， 熟练掌握 Flex 布局 ， 可以 独立 实现 复杂 样式 布局 ； 
 2 、 深入 理解 Javascript ， 熟悉 ES6 ， 熟练 使用 React ／ Vue 至少 一种 ， 有 ReactNative ／ Weex 等 Hybrid 开发 经验 加 分 ； 
 3 、 了解 js 模块化 开发 ， 熟练 使用 前端 构建 工具 webpack / gulp ， 掌握 常规 前端 优化 手段 ； 
 4 、 了解 一种 后 端 研发 语言 Node . js ／ Java ／ Python ， 理解 Restful   API ； 
 5 、 理解 HTTP 通讯 协议 ， 了解 HTTP / 2 ／ WebSocket ／ Speedy ； 
 6 、 对 机器 学习 有 一定 的 兴趣 ， 熟悉 canvas 、 webgl 等 数据 可视化 方法 ； 
 7 、 良好 的 团队 沟通 和 协作 能力 ， 能够 推动 客户 一起 完成 目标 。
